Both the Network Sniffer and Keylogger programs were developed and tested within a dedicated Python virtual environment in VS Code to ensure clean dependency management and project isolation. By utilizing a virtual environment, I ensured that the specific versions of the Scapy and pynput libraries remained independent of the global system Python installation, preventing potential version conflicts. Furthermore, because these tools interact directly with low-level system hardware—specifically the Network Interface Card (NIC) for packet capture and the Keyboard Human Interface Device (HID) for event listening—they require Root or Administrative privileges to execute successfully. This security requirement is a standard safeguard in modern operating systems to prevent unauthorized access to sensitive hardware data.
